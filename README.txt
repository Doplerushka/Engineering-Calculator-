Инженерный калькулятор на Python — это объектно-ориентированное приложение,
объединяющее в себе базовую арифметику, 
тригонометрию с углами в градусах, 
матричные операции (сложение, вычитание, умножение на скаляр, произведение матриц, транспонирование,
определитель, обратная матрица), численное интегрирование методом трапеций с возможностью регистрации пользовательских функций
и построение графиков произвольных выражений с использованием библиотек NumPy и Matplotlib.
Все вычисления сохраняются в единой истории, которая доступна для просмотра и очистки.
Архитектура построена на классах: History хранит записи, BaseCalculator отвечает за арифметику, TrigCalculator — за тригонометрию,
MatrixCalculator — за матрицы, IntegrationCalculator — за интегралы, GraphCalculator — за графики, 
а фасад EngineeringCalc объединяет их с общим объектом истории, обеспечивая единую точку доступа. 
Для запуска достаточно склонировать репозиторий, создать виртуальное окружение, активировать его,
установить зависимости из requirements.txt (numpy и matplotlib) и выполнить python engcalc.py. 
Демонстрационный скрипт выполнит проверку всех модулей:
покажет результаты сложения, вычитания, умножения, деления, возведения в степень, извлечения корня, 
вычислит синус, косинус, тангенс, котангенс, арксинус, арккосинус, арктангенс, 
сложит и перемножит матрицы, найдёт определитель и обратную матрицу,
проинтегрирует x² от 0 до 2 и sin(x) от 0 до π, а также построит графики трёх функций (x², sin(x), cos(x)). 
После этого будет выведена полная история операций, а затем она будет очищена. 
Проект подходит для учебных и инженерных целей, легко расширяется новыми операциями. 
Лицензия MIT, автор приветствует идеи и пул-реквесты. 
Для установки зависимостей используйте команду pip install -r requirements.txt, где файл содержит numpy и matplotlib. 
Всё просто, наглядно и работает из коробки.

1. History
Управляет историей всех выполненных операций в калькуляторе.
add_records(category, operation, result) — добавляет запись в историю.
show() — выводит на экран всю историю в формате 
clear() — полностью очищает историю.
2. BasedCalculator

Реализует базовые арифметические действия.
AddOperation(a, b) — сложение (\(a + b\)).
MinusOperation(a, b) — вычитание (\(a - b\)).
UmnohOperation(a, b) — умножение (\(a \cdot b\)).
DeleniaOperation(a, b) — деление (\(a / b\)). 
Защищено от деления на ноль.

3. Trigonometria
Работает с тригонометрическими функциями.

Примечание: Прямые функции принимают значения в градусах, обратные — возвращают результат в градусах.
sin(degrees) / cos(degrees) / tan(degrees) — синус, косинус и тангенс угла.
asin(value) / acos(value) / atan(value) — арксинус, арккосинус и арктангенс числа.

4. GraphCalculatorВизуализирует переданные математические выражения с помощью библиотеки matplotlib.
plot_expression(expression, x_min=-360, x_max=360) — строит график строкового выражения (например, 'sin(x) + cos(x)'). 
Автоматически сглаживает разрывы тангенса.

5. MatrixCalc (Наследует BasedCalculator)
Выполняет операции над двумерными списками (матрицами).
MatrixAdd(A, B) / MatrixMinus(A, B) — сложение и вычитание матриц одинакового размера(например, 'matrix_A = [[1, 2], [3, 4]]').
MatrixUmn(A, B) — матричное умножение (проверяет согласованность размерностей).
MatrixDel(A, b) — деление всех элементов матрицы на число \(b\).

6. IntegrationCalculator
Вычисляет определенный интеграл функции методом трапеций.
IntegrateOperation(func, a, b, n=1000) — интегрирует переданную Python-функцию (например, 'sqrt((2*pi)**2 - x**2)') на отрезке \([a, b]\) с количеством шагов разбиения n.

7. EngineeringCalc
Главный класс, объединяющий все калькуляторы в единый интерфейс со встроенной общей историей.
Включает модули: arithmetic, trigonometria, matrix, integration.q
show_history() — отображает общую историю операций.
